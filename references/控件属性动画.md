# 控件属性动画

MC 为部分 UI 控件属性提供了一套属性动画逻辑，使控件的某个属性值跟随时间发生变化，从而产生动画效果。

## 概述

属性动画以 JSON 数据的形式写在 UI 文件中，格式和控件类似。一个属性动画数据块的基本结构如下：

```json
{
    "anim_type": "alpha",
    "duration": 1.0,
    "from": 0.0,
    "to": 1.0
}
```

这表示控件在 1 秒内 alpha 值从 0 线性变化到 1。

### 支持的属性动画类型

当前支持动画的属性有：

| 动画类型 | `anim_type` 值 | 挂载属性 |
|----------|----------------|----------|
| 透明度 | `"alpha"` | `alpha` |
| 裁剪 | `"clip"` | `clip_ratio` |
| 颜色 | `"color"` | `color` |
| 序列帧 | `"flip_book"` | `uv` |
| 位移 | `"offset"` | `offset` |
| 尺寸 | `"size"` | `size` |
| UV动画 | `"uv"` | `uv` |

同一个控件可以有多个属性同时设置属性动画，所有的属性动画都会在界面创建后自动播放。

## JSON 写法

将动画绑定到控件上有两种写法。

### 写法一：内联写法

将动画数据块直接赋给对应属性：

```json
{
    "alphaImg": {
        "alpha": {
            "anim_type": "alpha",
            "duration": 0.30,
            "from": 0.0,
            "to": 1.0
        },
        "texture": "textures/netease/common/image/default",
        "type": "image",
        "visible": true
    }
}
```

### 写法二：引用写法（可复用）

将动画写成自定义控件形式放在 JSON 文件最外层，在属性值中以 `"@" + 动画名` 引用：

```json
{
    "alphaImg": {
        "alpha": "@show_alpha_ani",
        "texture": "textures/netease/common/image/default",
        "type": "image",
        "visible": true
    },
    "show_alpha_ani": {
        "anim_type": "alpha",
        "duration": 0.30,
        "from": 0.0,
        "to": 1.0
    }
}
```

两种写法效果相同，但引用写法可被多个控件复用。

## 通用属性

以下属性对所有属性动画类型通用：

| 属性 | 类型 | 说明 |
|------|------|------|
| `anim_type` | string | 动画类型，取值 `"alpha"` / `"clip"` / `"color"` / `"flip_book"` / `"offset"` / `"size"` / `"uv"` |
| `duration` | float | 动画持续时间，单位秒 |
| `from` | 取决于动画类型 | 动画起始值，控件初始化时会用该值初始化对应属性 |
| `to` | 取决于动画类型 | 动画结束值 |
| `next` | string | 当前动画播放完毕后接着播放的动画引用名（如 `"@next_ani"`） |

> 填写属性值时请注意不同动画所需的值类型和取值范围均不同，填写错误会导致动画无法生效或表现异常。

### 动画链与循环

通过 `next` 属性可以将多段动画串联播放。以下是一个 alpha 从 0→1 保持 1 秒后再 1→0 的渐变动画示例：

```json
{
    "alphaImg": {
        "alpha": "@show_alpha_ani",
        "texture": "textures/netease/common/image/default",
        "type": "image",
        "visible": true
    },
    "show_alpha_ani": {
        "anim_type": "alpha",
        "duration": 0.30,
        "from": 0.0,
        "next": "@hold_alpha_ani",
        "to": 1.0
    },
    "hold_alpha_ani": {
        "anim_type": "alpha",
        "duration": 1,
        "from": 1.0,
        "next": "@hide_alpha_ani",
        "to": 1.0
    },
    "hide_alpha_ani": {
        "anim_type": "alpha",
        "duration": 0.30,
        "from": 1.0,
        "to": 0.0
    }
}
```

若要实现循环，可以让链中最后一段动画的 `next` 指向链中某一段动画（注意：内联写法的动画不能被引用，因此无法参与循环）。

## 各动画类型详解

### 透明度动画 (alpha)

挂载在控件的 `alpha` 属性上。

| 属性 | 说明 |
|------|------|
| `anim_type` | `"alpha"` |
| `from` | 透明度起始值，取值 0 - 1.0 |
| `to` | 透明度结束值，取值 0 - 1.0 |

```json
{
    "alphaImg": {
        "alpha": "@show_alpha_ani",
        "texture": "textures/netease/common/image/default",
        "type": "image",
        "visible": true
    },
    "show_alpha_ani": {
        "anim_type": "alpha",
        "duration": 0.30,
        "from": 0.0,
        "next": "@hold_alpha_ani",
        "to": 1.0
    },
    "hold_alpha_ani": {
        "anim_type": "alpha",
        "duration": 1,
        "from": 1.0,
        "next": "@hide_alpha_ani",
        "to": 1.0
    },
    "hide_alpha_ani": {
        "anim_type": "alpha",
        "duration": 0.30,
        "from": 1.0,
        "to": 0.0
    }
}
```

### 裁剪动画 (clip)

挂载在图片控件的 `clip_ratio` 属性上。

| 属性 | 说明 |
|------|------|
| `anim_type` | `"clip"` |
| `from` | 裁剪程度起始值，0 = 不裁剪，1 = 完全裁剪（不渲染） |
| `to` | 裁剪程度结束值，取值 0 - 1.0 |

```json
{
    "clipImg": {
        "clip_ratio": {
            "anim_type": "clip",
            "duration": 1.0,
            "from": 0.0,
            "to": 1.0
        },
        "texture": "textures/netease/common/image/default",
        "type": "image",
        "visible": true
    }
}
```

### 颜色动画 (color)

挂载在图片控件的 `color` 属性上。

| 属性 | 说明 |
|------|------|
| `anim_type` | `"color"` |
| `from` | 颜色起始值，格式 `[r, g, b]`，各通道取值 0 - 1，不支持 alpha |
| `to` | 颜色结束值 |

```json
{
    "colorImg": {
        "color": "@color_ani",
        "texture": "textures/netease/common/image/default",
        "type": "image",
        "visible": true
    },
    "color_ani": {
        "anim_type": "color",
        "duration": 1.0,
        "from": [1, 0, 0],
        "to": [0, 0, 1]
    }
}
```

### 序列帧动画 (flip_book)

挂载在图片控件的 `uv` 属性上。通过切换图片显示区域实现逐帧动画。

| 属性 | 类型 | 说明 |
|------|------|------|
| `anim_type` | string | `"flip_book"` |
| `initial_frame` | int | 起始帧索引 |
| `frame_count` | int | 序列帧贴图中的总帧数 |
| `fps` | int | 每秒播放帧数 |
| `reversible` | bool | 默认 `false`；为 `true` 时播完后倒放回第一帧 |

```json
{
    "flipbookImg": {
        "texture": "textures/ui/my_eating_apple",
        "type": "image",
        "uv": "@flipbook_ani",
        "uv_size": [64.0, 64.0],
        "visible": true
    },
    "flipbook_ani": {
        "anim_type": "flip_book",
        "initial_frame": 10,
        "frame_count": 36,
        "fps": 10,
        "reversible": false
    }
}
```

**序列帧控件要求：**

| 属性 | 说明 |
|------|------|
| `texture` | 贴图路径，图片需为序列帧样式（多个等大子图拼成一张） |
| `uv` | 挂载序列帧动画引用 |
| `uv_size` | 每一帧的像素尺寸，不能为 `[0, 0]` |

> 序列帧贴图中每个子图的大小必须一致。
> 序列帧动画会自动循环，因此没有循环属性，且其后续 `next` 动画不会被播放。

### 位移动画 (offset)

挂载在控件的 `offset` 属性上。

| 属性 | 说明 |
|------|------|
| `anim_type` | `"offset"` |
| `from` | 位移起始值，格式 `[x, y]`，相对父节点的偏移 |
| `to` | 位移结束值 |

```json
{
    "offsetImg": {
        "offset": "@offset_ani",
        "texture": "textures/netease/common/image/default",
        "type": "image",
        "visible": true
    },
    "offset_ani": {
        "anim_type": "offset",
        "duration": 1.0,
        "from": [0, 0],
        "to": [0, 50]
    }
}
```

**注意：**
- 起始值和结束值的偏移类型必须相同。例如 `from` 为 `["50%", 0]` 时，`to` 也必须是百分比+像素形式如 `["100%", 50]`。
- 动态修改挂载了位移动画的控件尺寸时，应使用 `SetFullSize` 而非 `SetSize`。
- 动态修改偏移时，应使用 `SetFullPosition` 而非 `SetPosition`。

### 尺寸动画 (size)

挂载在控件的 `size` 属性上。

| 属性 | 说明 |
|------|------|
| `anim_type` | `"size"` |
| `from` | 尺寸起始值，格式 `[width, height]` |
| `to` | 尺寸结束值 |

```json
{
    "sizeImg": {
        "size": "@size_ani",
        "texture": "textures/netease/common/image/default",
        "type": "image",
        "visible": true
    },
    "size_ani": {
        "anim_type": "size",
        "duration": 1.0,
        "from": [100, 100],
        "to": [150, 150]
    }
}
```

**注意：**
- 起始值和结束值的类型必须相同（百分比与像素不能混用）。
- 动态修改尺寸时使用 `SetFullSize`，动态修改偏移时使用 `SetFullPosition`。

### UV 动画 (uv)

挂载在图片控件的 `uv` 属性上。

| 属性 | 说明 |
|------|------|
| `anim_type` | `"uv"` |
| `from` | UV 坐标初始值 `[x, y]`，以图片左上角为原点偏移像素 |
| `to` | UV 坐标结束值 |

```json
{
    "uvImg": {
        "uv": "@uv_ani",
        "texture": "textures/netease/common/image/default",
        "type": "image",
        "visible": true
    },
    "uv_ani": {
        "anim_type": "uv",
        "duration": 5,
        "from": [0, 0],
        "to": [2240, 0]
    }
}
```

## Python 控制动画接口

SDK 提供了以下接口在 Python 中动态控制属性动画：

| 接口 | 说明 |
|------|------|
| `RegisterUIAnimations` | 注册动画 |
| `UnregisterUIAnimation` | 取消注册动画 |
| `PauseAnimation` | 暂停动画 |
| `PlayAnimation` | 播放动画 |
| `StopAnimation` | 停止播放动画 |
| `SetAnimation` | 给单一属性设置动画 |
| `RemoveAnimation` | 给单一属性移除动画 |
| `SetAnimEndCallback` | 设置动画播放结束后的回调 |
| `RemoveAnimEndCallback` | 删除动画结束回调 |

其中 `RegisterUIAnimations` / `UnregisterUIAnimation` 属于通用接口，其余属于 UI 控件接口。详见 [全部自定义UI接口](./全部自定义UI接口.md) 和 [核心接口](./核心接口.md)。
